# FoodOrderV1 v1.5.0

Social management and trusted activity notifications.

## Highlights

- Remove an existing friend from both users' friend lists.
- Block duplicate invitations while a user is already active or pending in a group.
- Let group owners edit group details, remove members, and delete groups.
- Let non-owner members leave groups.
- Preserve independent direct bucket access when group-derived access is removed.
- Redesigned friends and groups cards with member status, owner controls, and clearer mobile layouts.
- Added a navbar notification center with unread badges and activity links.
- Added trusted notifications for social requests, group changes, bucket sharing/updates/deletion, and order placement/status/deletion.
- Added Functions, Firestore rules, and Playwright regression coverage.
