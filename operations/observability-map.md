# Observability

Current local diagnostics are user-visible errors, console errors in development, CI results, Firebase service consoles, and support evidence. Production analytics/error reporting is not configured. Any future telemetry requires privacy purpose, data minimization, redaction, retention, consent/legal review where applicable, sampling, access control, and user-facing documentation.
