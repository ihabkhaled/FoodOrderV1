# Privacy model

Processed data is limited to email/display name, locale/theme/currency preferences, user-authored buckets/items/prices, and user-authored orders/notes/status/timestamps. No sensitive-category inference, biometric processing, location, contacts, camera, microphone, advertising, or third-party analytics is currently implemented.
